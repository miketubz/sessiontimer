📋 Programme Timer — Help

➕ Add Programme

Adds a new row to the table. Each row represents one programme or segment with its own start time, end time, title, and timer.

↺ Reset Programme

Clears all start times, end times, and titles from every row. All timers are reset to idle state.

⏱ Start / Pause / Reset (per row)

Each row has its own controls. Set a start and end time first — the duration auto-calculates. Press Start to begin the countdown. You will be alerted 5 minutes before time is up with a sound and banner.

🤖 Autopilot

Automatically starts each row in sequence. Set an interval delay (1 second to 6 hours) between rows. Rows without a set time are skipped. Click Autopilot again while running to cancel or continue.

📲 Install App

Installs Programme Timer as a standalone app on your device (Android/Desktop). The Install button appears once your browser detects the app is ready to install.
